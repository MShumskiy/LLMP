✅ Solution: Allow Port 8000

    Open Windows Defender Firewall (Win + R → type firewall.cpl → hit Enter).
    Click Advanced settings (on the left).
    Go to Inbound Rules → New Rule.
    Select Port → Click Next.
    Choose TCP and enter 8000.
    Click Allow the connection → Next.
    Select Private Network (to allow only LAN connections).
    Name it FastAPI Access → Finish.
